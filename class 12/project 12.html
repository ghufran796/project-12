Positioning

A div with no position property behaves as a block-level element, taking up the full width of its container and positioning itself according to the normal document flow.


A div with position: sticky behaves like a relatively positioned element until it reaches a specified scroll position, after which it sticks to a defined position (like the top or bottom of its container) as the user scrolls. TRY SCROOLING IT SIDEWARDS


A position: relative element stays in the normal flow but can be offset using top, right, bottom, or left, without affecting surrounding elements' positions.


A position: absolute element is removed from the normal flow and positioned relative to its nearest positioned ancestor (not the document), using top, right, bottom, or left.


A position: fixed element is removed from the document flow and positioned relative to the viewport, staying fixed in place even when the page is scrolled TRY SCROOLING

Opacity

When an element has opacity: 0.1, it is almost fully
transparent, allowing most of the background to show through, but
the element is still faintly visible.


When an element has opacity: 0.2, it is mostly
transparent, with 20% visibility, meaning you can see it
faintly while most of the background shows through.


Setting an element's opacity to 0.3 in CSS
makes it 30% visible and 70% transparent. This means the
element will appear faint and allow more of the background
to show through compared to higher opacity values.


An opacity of 0.4 makes an element 40% visible and
60% transparent, creating a light, see-through effect.


An opacity of 0.5 makes an element 50% visible and
50% transparent, creating a semi-transparent effect. This allows
the background to be partially seen while still keeping the element recognizable.

An opacity of 0.6 makes an element 60% visible and 40% transparent,
giving it a fairly solid appearance while still allowing some background to show
through. This level of opacity is often used to create depth without fully obscuring
underlying content.


An opacity of 0.7 means the element is 70% visible and 30% transparent
, giving it a solid appearance while still allowing some background to show through.


An opacity of 0.8 means the element is 80% visible and 20%
transparent, making it quite solid while allowing just a hint of
the background to be visible. This level of opacity enhances visibility
while maintaining a subtle blend with the surrounding content.


An opacity of 0.9 means the element is 90% visible
and 10% transparent, making it nearly solid while allowing just
a tiny hint of the background to show through.

opaque and fully visible, with no transparency. This ensures
